# Downloading the replication package
The replication package is available as release -- the replication package is downloaded by clicking on the latest release (Replication package v1.0.0) under "Releases" (right side of this page) and downloading "replication.package.zip" by clicking the zip-file. The full instructions to replicate all tables and figures in Cavaliere, Mikosch, Rahbek and Vilandt (2026) are provided in the file README.pdf.
